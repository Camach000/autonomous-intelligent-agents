# 🤖 Autonomous Intelligent Agents: Pathfinding & Decision Making

> A simulation environment demonstrating the implementation of progressively complex AI architectures (Reactive, Deliberative, and Reinforcement Learning) for autonomous agents navigating dynamic environments.

## 🧠 About the Project

Developed as part of the Artificial Intelligence for Autonomous Systems (IASA) coursework, this project bridges core AI algorithms with robust software engineering principles. It features a "Prospector Agent" capable of target collection and obstacle avoidance in maze-like environments.

**Key Highlights:**
* **Architectural Complexity:** Programmed intelligent agents integrating Reactive (Subsumption), Deliberative (Search), and Reinforcement Learning architectures.
* **Algorithmic Implementation:** Applied state-space search algorithms (A*, BFS, Uniform Cost) and Sequential Decision Processes (MDPs) for dynamic trajectory planning.
* **Software Engineering:** Designed with modularity in mind, implementing Software Engineering patterns (MVC, State Machine) to ensure code robustness and scalability.

## 🛠️ Tech Stack

* **Languages:** Java, Python
* **Algorithms:** A*, Breadth-First Search (BFS), Uniform Cost Search, Markov Decision Processes (MDPs)
* **Architecture:** Model-View-Controller (MVC), State Machine, Subsumption Architecture

## 📂 Repository Structure

* `/java-agents`: Contains the core simulation of the "Prospector Agent" built in Java, highlighting the MVC and State Machine implementations.
* `/python-agents`: Python scripts focused on reinforcement learning and sequential decision-making models.
